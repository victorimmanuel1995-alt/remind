# Reminders

A lightweight, iPhone-style Reminders app in a single HTML file. No build step, no dependencies, no server. Everything is saved in your browser with `localStorage`.

## Features
- Lists with custom colour + icon
- Smart lists: Today, Scheduled, All, Flagged
- Add reminders with notes, date & time, flag, and priority (! / !! / !!!)
- Tap the circle to complete; completed items collapse into a section you can clear
- Search across all reminders
- Light and dark mode (follows your system)

## Run it locally
Just open `index.html` in any browser. That's it.

## Publish free with GitHub Pages
1. Create a new repository on GitHub and upload `index.html` (and this `README.md`).
2. Go to **Settings → Pages**.
3. Under **Build and deployment → Source**, choose **Deploy from a branch**.
4. Pick the `main` branch and the `/ (root)` folder, then **Save**.
5. Wait a minute — your app will be live at
   `https://<your-username>.github.io/<repo-name>/`

## Notes
- Data lives only in the browser you use it in — it isn't synced between devices.
- To reset everything, clear the site's storage in your browser settings.
